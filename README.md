# sales-data-analysis
Sales Data Analysis Project
📌 Project Overview

This project focuses on analyzing sales data to uncover insights related to customer behavior, revenue trends, profitability, and product performance.

Using SQL, Python, and Power BI, the dataset was cleaned, analyzed, and transformed into interactive visualizations to support business decision-making.

🎯 Objectives

The analysis aimed to answer key business questions such as:

Which customers generate the highest revenue?
Which regions and product categories perform best?
How do sales trends change over time?
Which products generate high sales but low profit?
What patterns can be identified in customer purchasing behavior?

🛠 Tools & Technologies Used
Python
Pandas
SQL
Power BI
Microsoft Excel
Matplotlib

Data Cleaning & Preparation
The dataset was cleaned and prepared using Python and Pandas.
Some preprocessing steps included:

Handling missing values
Correcting data types
Cleaning inconsistent text values
Formatting date columns
Removing unnecessary characters and symbols

📊 Exploratory Data Analysis (EDA)

SQL queries and Python analysis were used to explore:

Top customers by revenue
Sales performance by region and category
Profitability analysis
Shipping mode distribution
Customer purchase behavior
Moving average sales trends

📈 Dashboard & Visualizations

An interactive Power BI dashboard was created to visualize:

Sales trends over time
Top-performing customers
Regional sales distribution
Category and sub-category performance
Sales vs Profit scatter analysis
Moving average sales trends

Key Insights

A small percentage of customers contributed significantly to overall revenue.
Some product categories generated high sales but relatively low profits.
Certain regions consistently outperformed others in sales performance.
Moving averages helped identify long-term sales trends more clearly.
